# Atheena
A simple Inventory management CRM with live update on Dashboard and view detailed Transactions.

## Technologies
#### Backend 
- 🐹 Golang
- 🍃 MongoDB
- 🧠 Redis
- 🐳 Docker

#### Frontend
- 🟦 TypeScript
- ⚛️ React Framework
- 🎨 Chakra UI / Shadcn UI
- 🌬️ Tailwind CSS
